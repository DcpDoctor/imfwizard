Features

Standards-compliant IMF package creation, from essence to delivery.

📦

AS-02 MXF Wrapping

Wrap JPEG 2000 codestreams and PCM audio into SMPTE 2067-5 compliant MXF track files.

📋

CPL Generation

Generate Composition Playlists conforming to ST 2067-3 with proper segment and sequence structures.

🔐

Hash & Signing

SHA-1 hash computation for all assets with optional XML-DSIG signing for secure delivery.

🖥️

Desktop GUI

Tauri-based desktop application with drag-and-drop media import and visual timeline editing.

⚡

Fast CLI

High-performance C++23 command-line tool for scripting and pipeline integration.

🌐

Cross-Platform

Build and run on Linux, macOS, and Windows with full CI across all platforms.

Quick Start

Create your first IMP in minutes.

$ git clone --recurse-submodules https://github.com/DcpDoctor/imfwizard.git
$ cd imfwizard && mkdir build && cd build
$ cmake .. -DCMAKE_BUILD_TYPE=Release
$ make -j$(nproc)

$ ./imfwizard create \
    --title "My Feature Film" \
    --video /path/to/j2k_frames/ \
    --audio /path/to/audio.wav \
    --output /path/to/output_imp/

Roadmap

Building towards a complete IMF workflow.

Phase 1 — Current

OV Creator

J2K + WAV → AS-02 MXF track files, CPL, PKL, AssetMap. Valid App#2E Original Version IMP.

Phase 2

Supplemental IMP

Create supplemental versions referencing OV track files. Partial PKL/AssetMap generation.

Phase 3

Encoding Pipeline

Accept TIFF/EXR/DPX image sequences. Encode to J2K via grok. Optional ACES (App#5) support.

Phase 4

Full GUI

Timeline editor, drag-drop media import, supplemental creation workflow, validation integration.
